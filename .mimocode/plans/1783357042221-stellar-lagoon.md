# Plan: Add MP3 Player to new branch `player`

## Goal
Create branch `player` from `main`. Add MP3 playback: play/pause, next, previous track. The loaded file list doubles as the playlist.

## Prerequisite
```bash
sudo apt install qtmultimedia5-dev libqt5multimedia5-plugins
```
Add `-lQt5Multimedia` to Makefile LDFLAGS.

## New Files

### `include/mp3player.h` / `src/mp3player.cpp`
Wrapper around `QMediaPlayer` + `QMediaPlaylist`:
- `play()`, `pause()`, `stop()`, `next()`, `previous()`
- `setPlaylist(QList<MP3File*>)` — builds playlist from loaded files
- `setCurrentFile(MP3File*)` — jump to specific file
- `state()` — Playing/Paused/Stopped
- `position()` / `duration()` — for seek bar
- `setPosition(qint64)` — seek
- Signals: `stateChanged`, `positionChanged`, `durationChanged`, `currentFileChanged`

### No new files for UI — modify `mainwindow.h/cpp`

## Modified Files

### `include/mainwindow.h`
- Add `#include <QMediaPlayer>`, `#include <QSlider>`, `#include <QLabel>`
- Add forward decl: `class Mp3Player;`
- Add member: `Mp3Player *m_player;`
- Add UI widgets: `QPushButton *m_playBtn`, `QPushButton *m_prevBtn`, `QPushButton *m_nextBtn`, `QSlider *m_seekSlider`, `QLabel *m_timeLabel`
- Add slots: `onPlayPause()`, `onNext()`, `onPrevious()`, `onPositionChanged()`, `onDurationChanged()`, `onSeek(int)`, `onCurrentFileChanged()`

### `src/mainwindow.cpp`
**New player bar** (between toolbar and splitter):
```
[|◄] [▶ Play] [►|] [===seek slider===] [00:00 / 00:00]
```

- `onPlayPause()`: toggle play/pause. If stopped, play selected row's file.
- `onNext()`: select next row, play it.
- `onPrevious()`: select previous row, play it.
- Double-click on table row → play that file.
- `onPositionChanged()` / `onDurationChanged()` → update slider + time label.
- `onSeek()` → `m_player->setPosition()`.
- When files are added/removed, rebuild playlist.

### `Makefile`
- Add `mp3player.cpp` to SOURCES (auto via wildcard)
- Add `-lQt5Multimedia` to LDFLAGS
- Add `-I/usr/include/x86_64-linux-gnu/qt5/QtMultimedia` to INCLUDES

## Flow
1. User loads MP3 files → table populates → playlist built internally
2. User clicks a row → detail panel shows metadata
3. User clicks ▶ Play (or double-click row) → file plays
4. Seek slider tracks position, time label updates
5. Previous/Next buttons navigate the list
6. Play button toggles between ▶ Play and ⏸ Pause

## Verification
1. `make clean && make` — compiles with Qt5Multimedia
2. Run `./bin/MetadataMP3`, load MP3 folder
3. Double-click a file → should start playing, slider moves, time updates
4. Play/Pause button toggles correctly
5. Previous/Next navigate tracks
6. Metadata editing still works alongside playback
